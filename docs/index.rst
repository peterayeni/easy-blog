Introducing Django-easy-blog
=============================

A pluggable application that provides blog capabilities to your Django project.

**Documentation work in progress!**

Features:

1. Just one type of post: Stories.
2. Default theme based on `twitter-bootstrap <http://twitter.github.com/bootstrap/>`_.
3. Posts may be in Draft/Public status, and published in the future.
4. Posts categorized with tags, with `django-tagging <http://code.google.com/p/django-tagging/>`_.
5. Comments managed with `django-comments-xtd <http://packages.python.org/django-comments-xtd/>`_.
6. Support for inline media with `django-inline-media <http://packages.python.org/django-inline-media/>`_.
7. Multiple Authors.
8. Search capabilities with `django-haystack <http://packages.python.org/django-haystack/>`_, and more.

Run the demo project to see django-easy-blog in action.
